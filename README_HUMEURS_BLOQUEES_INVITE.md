# Correction humeurs invité

Quand l’utilisateur n’est pas connecté :

- les boutons/icônes des humeurs sont bloqués comme les liens protégés de la navbar ;
- un cadenas apparaît sur chaque humeur ;
- la carte de note ne s’ouvre pas ;
- le thème ne change pas ;
- un message demande à l’utilisateur de se connecter.

Fichiers modifiés :

- `vue/html/afficheHumeur.php`
- `vue/css/position-cercle-emojis.css`
- `vue/js/theme.js`
