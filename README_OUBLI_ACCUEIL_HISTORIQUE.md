# Correction oubli humeur accueil + historique

- Le bouton `⏰ J’ai oublié une émotion` est placé sur la page d’accueil, à côté de `🌸 Choisir mon humeur`.
- Le bouton n’apparaît pas dans la navbar.
- Il apparaît seulement quand l’utilisateur est connecté.
- Le formulaire d’oubli enregistre l’humeur dans `user_humeur`.
- Si une note est remplie, elle est aussi enregistrée dans `note` avec la même date.
- Après validation, l’utilisateur est redirigé automatiquement vers la page Historique.
- Un message confirme que l’émotion oubliée a été ajoutée automatiquement.
